# psy_site_petrov_sport

Лендинг спортивного психолога Валерия Петрова (Санкт-Петербург).
Одностраничный статический сайт без сборки: `index.html` + изображения, инлайн-CSS/JS.

**Боевой адрес:** https://petrov-sport.ru

## Структура
- `index.html` — вся вёрстка, стили и микроразметка
- `images/` — изображения, используемые на странице
- `tests/check_layout.py` — проверка адаптива (скриншоты + контроль горизонтального скролла на брейкпоинтах 320/375/768/1024/1440)

## Деплой
Загрузить `index.html` и папку `images/` в корень сайта на хостинге.

## Проверка вёрстки
```
python tests/check_layout.py
```
